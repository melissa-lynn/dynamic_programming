    Section 1.3 Tiling Problems

    Suppose we have a \(1\times n\) board that we wish to cover completely with tiles, without having any tiles overlap. The tiles that we have available are \(1\times 1\) blue tiles and \(1\times 2\) red tiles. How many ways can we tile our \(1\times n\) board using these tiles?
    🔗
    For example, let’s count how many ways we can tile a \(1\times 5\) board. The combinations are pictured below.
    🔗
    We can tile our board in the following ways:
    • 5 blue tiles.
      🔗
      🔗
    • 3 blue tiles, 1 red tile.
      🔗
      🔗
    • 2 blue tiles, 1 red tile, 1 blue tile.
      🔗
      🔗
    • 1 blue tile, 1 red tile, 2 blue tiles.
      🔗
      🔗
    • 1 red tile, 3 blue tiles.
      🔗
      🔗
    • 1 blue tile, 2 red tiles.
      🔗
      🔗
    • 1 red tile, 1 blue tile, 1 red tile.
      🔗
      🔗
    • 2 red tiles, 1 blue tile.
      🔗
      🔗
    This gives us a total of 8 ways to tile our board.
    🔗
    Now, how can we find the number of ways to tile an \(1\times n\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles? For large values of \(n\text{,}\) it would become very difficult to count the possibilities, so we would like a better way.
    🔗
    Here, we’ll use a common algorithmic design strategy: reduce the problem to smaller problems. This is done using the following key observation.
    🔗

    Insight 1.3.1. Key Insight to Tiling Problems.

    When we tile a \(1\times n\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles, every tiling will end with either a blue tile or a red tile.
    🔗
    🔗
    This means that to find the total number of tilings for a \(1\times n\) board, we find the number of tilings that end with a blue tile and add that to the number of tilings that end with a red tile.
    🔗
    How do we find the number of tilings that end with a blue tile? A blue tile fills in the last slot on the board, leaving \(n-1\) open slots for tiles. This means that the number of tilings that end with a blue tile is the same as the number of ways to tile a \(1\times (n-1)\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles.
    🔗
    Similarly, to find the number of tilings that end with a red tile, notice that a red tile fills in the last two slots on the board, leaving \(n-2\) open slots for tiles. This means that the number of tilings that end with a red tile is the same as the number of ways to tile a \(1\times (n-2)\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles.
    🔗
    Putting these together, if we write \(T(n)\) for the number of ways to tile a \(1\times n\) board, we have discovered the following relationship:
    \begin{equation*} T(n) = T(n-1) + T(n-2). \end{equation*}
    We can use this recursive relationship to compute the number of ways to tile a board, once we figure out our base cases.
    🔗

    Checkpoint 1.3.2. Base Case: \(n=1\).

    When \(n=1\text{,}\) how many ways are there to tile a \(1\times n\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles?
    🔗
    🔗

    Checkpoint 1.3.3. Base Case: \(n=2\).

    When \(n=2\text{,}\) how many ways are there to tile a \(1\times n\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles?
    🔗
    🔗
    When \(n=3\text{,}\) we can use \(T(3) = T(2) + T(1)\text{,}\) so the base cases \(n=1\) and \(n=2\) are sufficient.
    🔗
    Putting all of these together, we can use the following formulas to find the number of ways \(T(n)\) to tile a \(1\times n\) board with \(1\times 1\) blue tiles and \(1\times 2\) red tiles.
    • \(\displaystyle T(1) = 1\)
      🔗
      🔗
    • \(\displaystyle T(2) = 2\)
      🔗
      🔗
    • \(T(n) = T(n-1) + T(n-2)\) for \(n\geq 3\)
      🔗
      🔗
    We can use these to define a function to compute the number of tilings.
    🔗
    def num_tilings(n):
  if n <= 1:
    return 1
  if n == 2:
    return 2
  return num_tilings(n-1) + num_tilings(n-2)
